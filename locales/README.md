# Locale Files

날짜별 폴더에 언어 JSON 파일을 저장합니다.

## 구조

```
locales/
├── 2026-02-26/      ← 최초 업로드
│   ├── ko.json
│   ├── en.json
│   ├── ja.json
│   ├── zh.json
│   └── es.json
├── 2026-03-05/      ← 다음 주
│   └── ...
└── ...
```

최신 파일 = 폴더명 정렬 마지막 (YYYY-MM-DD 형식)

## 업로드 방법

1. `locales/` 폴더에서 **Create new file** 클릭
2. 파일명에 `2026-03-05/ko.json` 처럼 날짜/파일명 입력
3. 또는 날짜 폴더 들어가서 **Upload files**로 5개 드래그

## 보관 정책

- **4주간 보관** 후 폐기
